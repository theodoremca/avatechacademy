Frequently Asked Questions

What is Stutern?

Stutern is a skills development and job placement platform. We train and connect young African talents with long-term employment. We offer training classes that help you learn valuable skills that are relevant and valuable for the 21st century. Lastly, we also work with employers who want to hire candidates with digital skills.


What courses are offered?

We have the following courses:

  • Android Development
  • Frontend Development
  • Data Science & Machine learning
  • User Interface/User Experience Design
  • Software Development

What course is right for me?

While all the courses lead to relatively great careers, you must choose a track that aligns with your strengths, goals, and aspirations in life. Each program is immersive, having a strong reason will help you see it through successfully. Below are a few hints from us:

  • If you have an interest in design (graphic, fashion, etc.), possess any type of artistic skill (writing, photography, etc.), or love to interact with people, then, our UI/UX Design course might be ideal for you. UI/UX Designers are important in the product development phase because their work is what makes a user love a product. It borrows a lot from psychology even though the design aspect is what is more pronounced.

  • If you have some interest in design and programming but would rather spend more time on your computer than with your prospective users, then Frontend might be for you. Another way to know if the Frontend Development course is right for you is if you are the kind of person who prefers someone else to bother about the right color combination, the layout of the page, and how beautiful it should look. As a Frontend Developer, your core role is to write HTML, CSS, and JavaScript codes that will turn (i.e. render, you will hear that a lot) those user-centric and beautiful designs (ideally given to you by a UI/UX designer) into a webpage. Frontend Developers have some design skills but use it for coding web pages rather than designing prototypes.

  • Software Development skills typically take a longer time to develop than Frontend Development skills because it is a combination of both Frontend and Backend Development. A few things that can help you decide if this is the right course for you is if you have an interest in creating functional software products. Also, if you are comfortable dealing with abstract and theoretical things (you will be doing things whose result might not be visible on a webpage but through logs), then this might be the right one for you. Software Development gives you a chance to create software products that can be used by an unlimited number of people. Only a few professions give you that opportunity. It's rad!

  • If you are interested in building mobile applications and not web applications, then our Android course might be the best option for you. Our Android Development track requires that you have some knowledge of programming already. You don't need to have any work experience to join this course but some knowledge to complete the application task.

  • If you like Sherlock Holmes, Data Science might be it for you. While that sounds like a joke, it's not far from reality. To put it into context, if you have an interest or knowledge of research and numbers, then you might do well with Data Science. You should also be open to learning how to code. The course will teach you the core programming language you need to be successful in this field.
